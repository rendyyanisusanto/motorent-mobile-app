# MotoRent Mobile App

This is the mobile application frontend for MotoRent, built with Vue 3 and Vite.

## Requirements

- Node.js
- npm or yarn

## Setup & Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables (if any) by creating `.env` and `.env.local` files based on `.env.example`.

## Running the Application

```bash
# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```
